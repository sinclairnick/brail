# Quick start

This guide aims to get you up-and-running with Brail quickly, while briefly going over the core concepts. To jump right in, check out the [**starter project**](/#get-started).

> In this guide, we will use next.js as our underlying framework

## Setup Next.js

First, we need to setup a next.js project.

<Tabs items={["pnpm", "npm", "yarn"]}>
<Tab>
```bash
pnpm create next-app
```
</Tab>
<Tab>
```bash
npx create-next-app
```
</Tab>
<Tab>
```bash
yarn create next-app
```
</Tab>
</Tabs>

## Install Brail

With Next.js installed, we can now install Brail.

<Tabs items={["pnpm", "npm", "yarn"]}>
<Tab>
```bash
pnpm add brail @brail/react
```
</Tab>

<Tab>
```bash
npm i brail @brail/react
```
</Tab>

<Tab>
```bash
yarn add brail @brail/react
```
</Tab>
</Tabs>
