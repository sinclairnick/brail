# Quick start

This guide aims to get you up-and-running with Brail quickly, while briefly going over the core concepts. To jump right in, check out the [**starter project**](/#get-started).

## Installation

<Tabs items={["pnpm", "npm", "yarn"]}>
<Tab>
```bash
pnpm add brail @brail/react
pnpm add -D @brail/eslint-plugin # optional
```
</Tab>

<Tab>
```bash
npm i brail @brail/react
npm i -D @brail/eslint-plugin # optional
```
</Tab>

<Tab>
```bash
yarn add brail @brail/react
yarn add -D @brail/eslint-plugin # optional
```
</Tab>
</Tabs>
