# Quick start

This guide aims to get you up-and-running with Brail quickly, while briefly going over the core concepts.

## Installation

<Tabs items={["pnpm", "npm", "yarn"]}>
<Tab>
```bash
pnpm add brail @brail/react @brail/devtools
pnpm add -D @brail/eslint-plugin
```
</Tab>

<Tab>
```bash
npm i brail @brail/react @brail/devtools
npm i -D @brail/eslint-plugin
```
</Tab>

<Tab>
```bash
yarn add brail @brail/react @brail/devtools
yarn add -D @brail/eslint-plugin
```
</Tab>
</Tabs>
